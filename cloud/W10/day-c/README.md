# W10 - Day C: Platform Integration & SRE Operations

## Nội dung học

- Tích hợp toàn bộ hệ thống từ W8 đến W10 (GitOps, Observability, Canary, Secrets, Security)
- Phân hoạch tài nguyên cụm dùng ResourceQuota và LimitRange
- Khái niệm Chaos Engineering trên Kubernetes và các công cụ (Chaos Mesh, LitmusChaos)
- SRE Incident Response Playbook 6 bước
- SRE Runbook mẫu khắc phục sự cố CrashLoopBackOff do lỗi ESO Secrets Sync
- Tối ưu hóa chi phí với AWS Cost Anomaly Detection và Budget Alerts
- Thiết lập Terraform IaC cho Cost Anomaly Monitor & Subscription

## Mục tiêu

Hiểu phương pháp tích hợp liên thông các công nghệ để bootstrap một secure platform dưới 2 giờ, thiết lập các chính sách giới hạn tài nguyên và xây dựng quy trình ứng phó sự cố (Runbook/Chaos) cùng tối ưu chi phí (FinOps).

- Hiểu mô hình tích hợp hệ thống GitOps + Canary + Observability + Security
