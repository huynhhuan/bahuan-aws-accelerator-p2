# Grafana

## Grafana là gì?

Grafana là nền tảng trực quan hóa dữ liệu.

## Chức năng

- Dashboard
- Alerting
- Data Visualization

## Kết nối dữ liệu

Grafana có thể đọc từ:

- Prometheus
- Loki
- Elasticsearch
- CloudWatch

## Dashboard phổ biến

### Infrastructure Dashboard

- CPU
- Memory
- Disk

### Application Dashboard

- Requests
- Errors
- Latency

## Lợi ích

Giúp vận hành dễ dàng thông qua biểu đồ trực quan.
