# Burn Rate Alert

## Burn Rate là gì?

Burn Rate thể hiện tốc độ tiêu thụ Error Budget.

## Error Budget

Nếu SLO là:

99.9%

Thì Error Budget là:

0.1%

Hệ thống được phép lỗi 0.1%.

## Ví dụ

SLO:

99.9%

Error Budget:

0.1%

Nếu lỗi tăng nhanh:

0.05% chỉ trong vài giờ

=> Burn Rate cao

=> Cần cảnh báo sớm

## Multi Window Burn Rate

Fast Window

- 5 phút
- 1 giờ

Slow Window

- 30 phút
- 6 giờ

## Mục tiêu

Phát hiện:

- Sự cố đột ngột
- Sự cố kéo dài

trước khi hết Error Budget.
